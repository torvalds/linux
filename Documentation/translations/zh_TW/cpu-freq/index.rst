=======================================================
Linux CPUFreq - Linux(TM)內核中的CPU頻率和電壓升降代碼
=======================================================

Author: Dominik Brodowski  <linux@brodo.de>

      時鐘升降允許你在運行中改變CPU的時鐘速度。這是一個很好的節省電池電量的方法，因爲時
      鐘速度越低，CPU消耗的電量越少。
