Allwinner sunXi Platforms Device Tree Bindings

Each device tree must specify which Allwinner SoC it uses,
using one of the following compatible strings:

  allwinner,sun4i-a10
  allwinner,sun5i-a10s
  allwinner,sun5i-a13
  allwinner,sun5i-r8
  allwinner,sun6i-a31
  allwinner,sun7i-a20
  allwinner,sun8i-a23
  allwinner,sun8i-a33
  allwinner,sun8i-a83t
  allwinner,sun8i-h3
  allwinner,sun9i-a80
